CUDA Execution Flow

How is a CUDA Program Run on GPU?

  • Step 1
    Load Data into CPU Memory
  • Step 2
    Copy and allocate Data to GPU Memory (Host-to-Device)
  • Step 3
    Execute GPU Kernel (Parallel Processing)
  • Step 4
    Synchronize and copy Results to CPU Memory (Device-to-Host)
  • Step 5
    Use Results on CPU
